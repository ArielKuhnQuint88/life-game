# Segurança

Não publique vulnerabilidades ou dados sensíveis em issues públicas. Entre em contato diretamente com o responsável pelo projeto.

O protótipo e os documentos atuais não devem ser usados para armazenar dados reais de saúde. Qualquer versão com usuários reais dependerá de revisão de segurança, privacidade e conformidade com a LGPD.
